Using Classes and IDs as Selectors

Think you're limited to using only HTML elements as selectors? Think again.

So far, we've only discussed creating style rules for HTML elements. But using classes and IDs to create style rules allows For example, if you create a style rule for the p paragraph tag, this will apply to all paragraphs in your document, and that might not be what you want. Classes and IDs are special selectors that can be applied to a subset of HTML elements

For example, the first paragraph in this section ("Think you're limited...") is a paragraph, but the style is different than the other paragraphs. That's because this paragraph has been assigned the class name of "lead."

  <p class="lead">
    Think you're limited to using only HTML elements as selectors? Think again.
  </p>

This allowed me to assign CSS rules that only apply to elements that have the class name of "lead".
