Doing Something

Most Ruby Tutorials run through language concepts with no end goal in mind. This tutorial is not that kind of tutorial.

This tutorial will allow you to change a list of your current semester's classes into part of an HTML file that you can use for your personal page.

To accomplish this, you will need to:

  • Create a comma-separated value file
  • Read the contents of that CSV file into your program
  • Separate the comma-separated values into separate stored values
  • Write the stored values into an HTML table

Let's begin with creating the CSV file.
