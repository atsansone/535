What Should I Learn?

After completing this section, you should understand how to:

  • Read Ruby code
  • Understand general programming terms
  • Identify:
    • Operators
    • Variables
    • Arrays
    • Hashes
    • Methods
    • Objects
    • Classes
  • Create and assign values to:
    • Variables
    • Arrays
    • Hashes
  • Create
    • Method
    • Class
    • Object
  • Error check your program

How we will do this is by creating a simple program that will let you create a list of your courses for the current semester and add them to a web page.
