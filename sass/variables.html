An Introduction to Sass

Here are some examples of why you'll want to get to know Sass:

Variables

This is usually where people get hooked on Sass: the ability assign variables to items. My favorite place to use them is when declaring colors in Hex or RGB mode: it's difficult to remember what #4C1BE0 means when you look through your CSS rules, and if you haven't been good about using comments, then you have to look it up manually.

Here is an example of using variables for color in Sass. This is what the normal CSS that is generated by Sass would look like:

body {
  background: #89a194;
  padding: 20px 10px 10px 10px;
  color: #322938;
}
body a {
  color: #a14016;
  font-weight: 900;
}

And here's how the Sass language makes that much more readable:

$backgroundColor: #89A194;
$bodyFontColor: #322938;
$linkColor1: #A14016;

body {
  background: $backgroundColor;
  padding: 20px 10px 10px 10px;
  color: $bodyFontColor;
}
body a {
  color: $linkColor1;
  font-weight: 900;
}
